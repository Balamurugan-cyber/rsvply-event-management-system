A modern, intelligent event management platform that streamlines RSVP collection and guest management with AI-powered insights for event organizers.

## 🌟 Features

### Core MVP Features
- ✅ **Event Creation** - Organizers can create events with name, date, venue, and description
- ✅ **Public RSVP Page** - Guests can RSVP with Attending/Not Attending/Maybe status
- ✅ **Optional Fields** - Meal preferences and messages to organizers
- ✅ **Organizer Dashboard** - Comprehensive view of all guest responses
- ✅ **Guest Filtering** - Filter by RSVP status (All/Attending/Not Attending/Maybe)
- ✅ **CSV Export** - Download complete guest data for external processing

### Enhanced Features
- 🤖 **AI Sentiment Analysis** - Automatic analysis of guest messages
- 📊 **Predictive Analytics** - Attendance rate and engagement predictions
- 💬 **Smart Response Suggestions** - AI-generated response templates
- 🔐 **Role-Based Authentication** - Separate flows for organizers and guests
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- 🎨 **Modern UI/UX** - Clean, intuitive interface with gradient designs
